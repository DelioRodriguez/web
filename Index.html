Proyectos

Shop Management System - Microservices Architecture

Este proyecto es un sistema de gestión de tienda desarrollado en C# utilizando .NET, diseñado para manejar la administración de productos, categorías y proveedores. Implementado con una arquitectura de microservicios, el sistema está compuesto por APIs creadas y consumidas en la misma tecnología. El frontend también se desarrolló en .NET, asegurando una integración fluida y eficiente.
